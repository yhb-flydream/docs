# box-shadow

向框添加一个或多个阴影

```css
box-shadow: h-shadow v-shadow blur spread color inset;

或者

box-shadow: h-shadow v-shadow blur spread color inset, h-shadow v-shadow blur spread color inset;
```

- h-shadow
  - 必需。水平阴影的位置。允许负值。
- v-shadow
  - 必需。垂直阴影的位置。允许负值。
- blur
  - 可选。模糊距离。
- spread
  - 可选。阴影的尺寸。
- color
  - 可选。阴影的颜色。
- inset
  - 可选。将外部阴影 (outset) 改为内部阴影。
