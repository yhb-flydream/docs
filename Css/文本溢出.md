# 文本溢出

## 单行文本溢出

```css
#box {
    width: 150px;
    overflow: hidden;
    white-space: nowrap;
    text-overflow: ellipsis;
}
```

## 多行文本溢出

```css
#box {
    width: 150px;
    display: -webkit-box !important;
    overflow: hidden;

    text-overflow: ellipsis;
    word-break: break-all;

    -webkit-box-orient: vertical;
    /*显示两行，如果显示3行再溢出则改写成 -webkit-line-clamp:3;*/
    -webkit-line-clamp: 2;
}
```

## 问题

- 火狐下不支持溢出省略号显示，建议使用

```css
max-height: --px;
max-width: --px;
overflow: hidden;
```
