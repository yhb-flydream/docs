# [attribute]

## `[attribute]`

所有带有 href 属性的元素

```js
$('[href]')
```

## `[attribute=value]`

所有 href 属性的值等于 "#" 的元素

```js
$("[href='#']")
```

## `[attribute!=value]`

所有 href 属性的值不等于 "#" 的元素

```js
$("[href!='#']")
```

## `[attribute$=value]`

所有 href 属性的值包含以 ".jpg" 结尾的元素

```js
$("[href$='.jpg']")
```
