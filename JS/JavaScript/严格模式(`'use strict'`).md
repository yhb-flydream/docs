# 严格模式(`'use strict'`)

- **1、全局模式**
  - **需在代码最前面添加`'use strict'`**
  - **严格模式下，代码编写一定要按照规则**
    - 必须用 var 定义变量
    - 一个对象不能有重复发属性名
    - 函数形参不能相同
    - eval 会产生单独做用域，避免全局变量污染
    - 无法在非函数代码块中声明函数
    - 严格模式下，禁止使用 with 语句
    - 函数的 caller 不能使用(不能再返回调用函数的函数)
    - arguments 的 callee 不能使用(不能再返回函数自身)
- **2、局部模式**
  - **在函数内部的第一行添加`'use strict'`**
  - 有了全局就不用在用局部

```javascript
function fn() {
    'use strict';
    ...
}
```
